1)->DDL -> DATA DEFINITION LANGUAGE -> MAINTAINS THE STRUTURE OR SCHEMA OF THE TABLE OR DATABASE OBJECTS-> "HERE WE USE KEYWORD TABLE"

CMD -> CREATE,ALTER,DROP,TRUNCATE

2)->DML -> DATA MANIPULATION LANGUAGE -> DATA INSTANCES -> WORK ON DATA  -> "HERE WE USE DIRECTLY TABLE NAME INSTEAD OF KEYWORD TABLE"

CMD -> INSERT,UPDATE,DELETE,SELECT

3)->DCL -> AUTHORIZATION -> ADMINISTRATOR CAN GIVE PRIVILEGES TO THE USERS -> ACCESS RIGHTS OF USERS ON TABLES

CMD -> GRANT,REVOKE

4)TCL -> TRANSACTION MANAGEMENT

CMD -> COMMIT,ROLL BACK

->QUERY COMMAND -> USED TO "RETRIEVE" THE RECORDS OF THE TABLE  -> FETCH THE DATA

CMD -> SELECT


->COLOUMNS ARE KNOWN AS FIELDS
